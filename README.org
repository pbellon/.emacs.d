#+TITLE:  Emacs configuration
* Global structure

- ~early-init.el~ early initialisation, contains optimisations for emacs
- ~init.el~ entry point of the emacs configuration, holds customized variables & loading of ~setup.el~
- ~setup.el~ main configuration file

* Used packages


|--------------+------------------------------------------+-----------------|
| Package name | Description                              | Main keybinding |
|--------------+------------------------------------------+-----------------|
| magit        | Git integration in emacs                 | ~C-c C-g~       |
| ace-window   | Navigation between frames                | ~C-x o~         |
| ivy + swiper | Search within buffer                     | ~C-s~           |
| doom-themes  | Customize emacs look with lots of themes | -               |
|--------------+------------------------------------------+-----------------|

|-----------------------------------+------------+------------------------------------------------------------------|
| Family                            | Keybinding | Function                                                         |
|-----------------------------------+------------+------------------------------------------------------------------|

| Common code edition               |            |                                                                  |
|-----------------------------------+------------+------------------------------------------------------------------|

|-----------------------------------+------------+------------------------------------------------------------------|
|                                   | ~C-x B~    | Open all buffer listing                                          |

|                                   | ~C-x o~    | Triggers ~ace-window~, replace default binding of ~other-window~ |
|                                   | ~C-x k~    | Kill a buffer                                                    |
